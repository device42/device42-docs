## Current Supported Languages

Device42 currently supports the following languages. Select your preferred language from the language selection dropdown menu:

- Dutch
- English
- French
- German
- Japanese
- Korean
- Polish
- Portuguese
- Russian
- Simplified Chinese
- Spanish
- UK English

If you have suggestions to improve a particular translation or would like additional support, please open a support ticket from the Main Appliance. We're happy to help!
