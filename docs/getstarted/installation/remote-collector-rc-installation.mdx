The Remote Collector (RC) is shipped as a virtual appliance and can be downloaded from our [**Autodiscovery** software download page](https://www.device42.com/autodiscovery/).

## Installation Instructions

The [**Installation and Configuration**](/auto-discovery/remote-collector-rc/#installation-and-configuration.mdx) section of the [**Remote Collector**](/auto-discovery/remote-collector-rc.mdx) page provides detailed instructions for installing an RC.
