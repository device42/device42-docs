---
title: "DBB - Cookbook"
---

The Data Building Blocks (DBB) Cookbook is a collection of queries that provide report examples for common use cases. These recipes feature DBB materialized views designed for quick reporting with little-to-no extra view joining required. The queries can be used as-is or modified to meet your business reporting requirements.

DBB views are materialized database views composed with customer-specific use cases in mind, and are refreshed on a nightly basis. Because DBB reporting requires access to all Device42 data, users must be granted the `Data Building Blocks` privilege to access DBB views.

The recipes are organized into three categories:

- [**Application Dependency Mapping Recipes**](/reports/dbb-cookbook/app-dependency-mapping)
- [**Compute Recipes**](/reports/dbb-cookbook/compute)
- [**Security Recipes**](/reports/dbb-cookbook/security)

Use the sidebar to browse individual recipes.
