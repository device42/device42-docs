title: "Devices with No OSs"
sidebar_position: 26
---

Returns devices with no OS captured — use this to identify opportunities to enrich your data with a full device discovery.

<details>
     <summary>Click to expand the code block</summary>
     ```sql
     select
          ds.device_name
          ,device_fk
          ,ds.device_type
          ,ds.physicalsubtype_name
          ,ds.serial_number
          ,ds.device_service_level
          ,ds.tags
          ,ds.device_first_added
          ,ds.device_last_changed
          ,ds.is_missing_os
          ,ds.ip_address
     from view_dbb_security_v2 ds
     where ds.is_missing_os = 'Yes' and ds.device_fk  is not null
     ```
</details>
