Compute Devices with Listener / Client mismatch of Prod and Non-Prod Service Levels.

<details>
    <summary>Click to expand the code block</summary>
    ```sql
    select distinct
         ds.device_fk
         ,ds.device_name
         ,ds.device_service_level
         ,ds.mismatched_listener_devices
         ,ds.mismatched_client_devices
    from view_dbb_security_v2 ds
    where ds.mismatched_listener_devices is not null
    or ds.mismatched_client_devices is not null
    order by 1
    ```
</details>

**NOTES**

Service Levels are defined by customer and not directly discovered.
