A run book is a comprehensive export of your Device42 CMDB data, delivered as an Excel spreadsheet with data organized into sheets by object category.

## Generate a Run Book

Navigate to **Analytics > Reports > Generate Run Book** to create a run book. The report takes a few minutes to generate, depending on the size of your CMDB.
