The Periodic Jobs page also lets you to view and manage periodic sampling performed on objects by discovery jobs. Find the Periodic Jobs list page under **Analytics > Discovery Status > Periodic Jobs**. 

You can quickly disable Resource Utilization for PDU items, metrics for Storage Arrays as well as SP and DB sampling using the **Actions** menu options.
