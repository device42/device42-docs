You can customize certain aspects of the Device42 look-and feel...

Navigate to Tools >> Settings >> Global Settings

At the bottom of the page, you will see two options...

![customization.png](/assets/images/2015-10-10-customization.png)

### Add your company logo to Device42

If you upload a logo, it will replace the Device42 logo on the login screen and through the application on the menu bar next to the user name...

![customization.png](/assets/images/2015-10-10-customization-2.png)

### Add user instructions to the login page

If you upload user instructions they will appear on the login page as shown below...

![customization.png](/assets/images/2015-10-10-customization-3.png)
