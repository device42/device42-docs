---
title: "Appliance Manager"
---

The Device42 Appliance Manager runs on port 4242 (HTTP by default, with HTTPS available to enable from inside) and provides administrative functions such as backups and restores, HTTPS certificate management, application restart and stop controls, and log or SNMPwalk generation for support. This section covers the topics below.
