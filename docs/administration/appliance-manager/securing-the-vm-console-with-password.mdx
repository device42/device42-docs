title: "Securing the VM console with password"
sidebar_position: 15
---

Starting with v4.3.5, you can secure the VM console menu with a password.

### Setting the password from the appliance manager

![Setting_the_password_from_the_appliance_manager.png](/assets/images/Setting_the_password_from_the_appliance_manager.png)

From System > Set VM Menu password, you can set the password for the console menu.

### New menu option to return to password prompt

![New_menu_option_to_return_to_password_prompt.png](/assets/images/New_menu_option_to_return_to_password_prompt.png)

Option #8 will be added if you enable password for the console.

### Password prompt screen

![Password_prompt_screen.png](/assets/images/Password_prompt_screen.png)

You will have to enter the password before you can the menu.
