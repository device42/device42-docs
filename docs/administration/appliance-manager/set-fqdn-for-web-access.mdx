---
title: "Set FQDN for Web Access"
sidebar_position: 16
---

You can limit access to the Device42 Main Appliance to a specific FQDN. Use of this setting will deny access to Device42 via IP address or other hostname and/or domain.

[Log in to the Appliance Manager](device42-appliance-manager-login.mdx), then navigate to **global Settings > Set Hostname for Web Console**. 

Enter the FQDN you want to use, check **Enable Hostname Checking** click **Submit**.
