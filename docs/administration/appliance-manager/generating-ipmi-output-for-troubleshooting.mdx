You can collect IPMI output for a device right from the GUI and send us the output for further troubleshooting.

### Collect IPMI output

![Collect IPMI output](/assets/images/2016-04-07-Generate-IPMI-output-01.png)

Go to: Application > Generate IPMI output

Add the FQDN or IP address, name and password, click generate, wait as it might take few minutes and send us the file it generates.
