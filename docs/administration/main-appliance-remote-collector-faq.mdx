This page answers frequently asked questions about downloading, updating, and managing the Device42 Main Appliance (MA) and Remote Collector (RC).

## Download Files

Go to [https://www.device42.com/update/](https://www.device42.com/update/) to download the latest Main Appliance (MA) or Remote Collector (RC). Enter your email address and Device42 will send you the file along with release notes and next steps.

## RC and MA Version Compatibility

You must be on the immediately prior Main Appliance release before downloading the latest RC update. For example, if the latest MA version is 17.07.00, you cannot download RC release 17.07.02 until you have upgraded to MA 17.07.00.

### Download the RC Before Installing the Newest MA

If you attempt to download the RC before installing the latest MA, the download will stop and prompt you to upgrade the MA first.

### Install an Older RC Update on a Newer MA Version

This will fail — the installer will prompt you to download the correct version.

## Update Remote Collectors with an MA Update

When you download an MA update, it will also update your Remote Collectors to the latest version.

## Upgrade from an Older MA Version

If you are more than three or four versions behind the current Main Appliance release, contact support for upgrade assistance. The Device42 update page shows the specific cutoff version.

If you are fewer than three versions behind, downloading the new MA file will bring you fully up to date.

## Update Remote Collectors on the Latest MA Release

Push the update via the Appliance Manager once you have the upgrade file. For step-by-step instructions, see the [Device42 FAQs](/getstarted/using-device42/faqs).

## Configure the Remote Collector Proxy

The proxy configured on the MA is also used by the Remote Collector. You can view and edit the inherited proxy settings from the [RC view and edit page](/auto-discovery/remote-collector-rc/#view-and-edit-remote-collectors).
