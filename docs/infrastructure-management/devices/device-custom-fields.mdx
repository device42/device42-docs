Custom fields let you store additional information on devices beyond the built-in fields. You can also choose to display custom fields in device pop-up panels and on layout pages for quick reference.

When adding or editing a device custom field, select **Include in Context Pop-ups** to display the field in pop-up panels and layout pages. For general information about custom fields, see [Custom Field Key Value Pairs](/administration/custom-key-value-pairs-explained.mdx).
