Archiving lets you retire devices and resources from active service while keeping a record of the object. Archived objects are stored on a separate list page where you can view, edit, or delete them.

This page explains how to archive devices and resources individually or in bulk, and how to manage archived objects.

## Archive a Device

1. Navigate to **Resources > Compute > All Devices** to display the Devices list page.
2. Click the name of the device you want to archive to display its view page, then click **Archive**.

    <ThemedImage
    alt="Archive device button"
    sources={{

        dark: useBaseUrl('/assets/images/archiving-devices-and-resources/archive-button-dark.png'),
    }}
    />

3. Click **Yes, I'm sure** to confirm. The device is added to the Archived Objects list page.

    <ThemedImage
    alt="Archive confirmation prompt"
    sources={{

    }}
    />

### Bulk Archive Devices

To archive multiple devices at once, select the devices from the Devices list page and choose **Fast Background Archive** from the **Actions** menu. This command does not display a confirmation prompt.

  alt="Bulk archive option"

## Archive a Resource

1. Navigate to **Resources > Resources > All Resources** to display the Resources list page.
2. Click the name of the resource you want to archive to display the resource view page.
3. Click **Archive** at the top right of the view page.

    <ThemedImage
    alt="Resource archive button"
    sources={{
        light: useBaseUrl('/assets/images/archiving-devices-and-resources/archive-resources-button-light.png'),
        dark: useBaseUrl('/assets/images/archiving-devices-and-resources/archive-resources-button-dark.png'),
    }}
    />
4. An **Archive Object** pop-up displays information about the resource. Click **Show** to see additional details, and add optional notes if needed.

    <ThemedImage
    alt="Archive resource popup"
    sources={{
        light: useBaseUrl('/assets/images/archiving-devices-and-resources/archive-resources-popup-light.png'),
        dark: useBaseUrl('/assets/images/archiving-devices-and-resources/archive-resources-popup-dark.png'),
    }}
    />

5. Click **Archive** to confirm. The resource is added to the Archived Objects list page.

Navigate to **Analytics > Audit Logs > Archived Objects** to display the list page.

## Delete an Archived Object

From the Archived Objects list page, select the checkbox next to the object you want to delete. Choose **Delete with Detailed Confirmation** or **Fast Background Delete** from the **Actions** menu.
