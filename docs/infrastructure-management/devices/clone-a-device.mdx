Cloning a device saves time and provides consistency when adding multiple similar devices. You select a source device, choose which properties to copy, and specify how many clones to create.

Navigate to **Tools > Templates and Bulk Operations > Clone Device**.

1. Select a **Device to clone** and choose the properties to include using the checkboxes. **Notes** are not included in clones.

    <ThemedImage
    alt="Device clone name and properties"
    sources={{
        light: useBaseUrl('/assets/images/clone-a-device/name-and-properties-light.png'),
        dark: useBaseUrl('/assets/images/clone-a-device/name-and-properties-dark.png'),
    }}
    />

2. Enter the number of clones in the **# of devices** field. Name the clones by selecting a **Device Name Profile**, or enter a comma-separated list of names to name each clone individually.

    <ThemedImage
    alt="Number of clones and naming options"
    sources={{
        light: useBaseUrl('/assets/images/clone-a-device/count-and-name-light.png'),
        dark: useBaseUrl('/assets/images/clone-a-device/count-and-name-dark.png'),
    }}
    />

3. Optionally, set an **Asset Number Profile**, **Service Level**, or **Customer** for the clones. If you apply a new **Service Level** or **Customer**, deselect the corresponding checkbox under **Properties** to prevent the original value from being copied.

4. Click **Clone** in the bottom-right corner to create the clones.

    alt="Additional properties and Clone button"

        light: useBaseUrl('/assets/images/clone-a-device/other-properties-light.png'),
        dark: useBaseUrl('/assets/images/clone-a-device/other-properties-dark.png'),
    }}
    />
