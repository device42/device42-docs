The Software In Use page shows individual instances of each software component across your devices. Unlike the All Software Components page — which lists each component once — this view shows one row per device-software combination, so the same software can appear multiple times if it is installed on multiple devices.

Navigate to **Resources > Software Components > Software In Use** to see these instances. You can filter by the **Name** of the software component, the **Device**, or the **End User**, and sort the results by **Device** or **End User**.

Click the software **Name** to view or edit the software-in-use entry.

## Software In Use Actions

Select one or more software-in-use entries, choose an action from the **Actions** dropdown, and confirm the dialog.

## Add a Software In Use Entry

Click **Create** to add a new software-in-use entry.

<ThemedImage
  alt="Add software in use form"
  sources={{
    light: useBaseUrl('/assets/images/software-in-use/software-in-use-add-new-light.png'),
    dark: useBaseUrl('/assets/images/software-in-use/software-in-use-add-new-dark.png'),
  }}
/>

Click **Software** to search for and select a software component already in Device42. Click the **plus icon** to add a new component from this dialog. If necessary, enter or update the **Raw License Count**, then click **Save** to add the entry.
