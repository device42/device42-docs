On-premises (on-prem) databases are non-cloud database instances identified and returned by Device42 discovery jobs. You can view the discovered on-prem database details, edit fields, and manage related resources.

This page covers how to view, edit, delete, and associate on-prem databases with Business Services.

## View On-Prem Databases

Select **Resources > Databases > On-Prem Databases** from the main menu.

<ThemedImage
  alt="On-Prem Databases menu"

    dark: useBaseUrl('/assets/images/on-premise-databases/on-prem-menu-19.06-dark.png'),
  }}
/>

Use the list page filters to find specific databases:

- **Search by resource name, host, database type**
- **Database Type** filter
- **More Filters** for additional filter options

<ThemedImage
  alt="On-Prem Databases list page"
  sources={{
    light: useBaseUrl('/assets/images/on-premise-databases/on-prem-filters-light.png'),
    dark: useBaseUrl('/assets/images/on-premise-databases/on-prem-filters-dark.png'),
  }}
/>

Click an on-prem database name to see its details.

<ThemedImage
  alt="On-Prem database details"

## Edit On-Prem Databases

From the database details page, click **Edit** to modify the following fields: **Database Instance Application Component**, **Custom Fields**, **Notes**, **Tags**, and **Service Level**.

<ThemedImage
  alt="Edit on-prem database fields"
  sources={{

    dark: useBaseUrl('/assets/images/on-premise-databases/edit-custom-fields-dark.png'),
  }}
/>

## Delete On-Prem Databases

Select one or more database instances on the on-prem databases list page. From the **Actions** dropdown menu, select **Fast Background Delete**, **Fast Background Archive**, or **Delete with Detailed Confirmation**.

<ThemedImage
  alt="On-Prem Databases action menu"
  sources={{

    dark: useBaseUrl('/assets/images/on-premise-databases/delete-actions-19.06-dark.png'),
  }}
/>

Confirm or cancel the deletion when prompted.

## Add an On-Prem Database to a Business Service

You can add an on-prem database to an existing or new [Business Service](/apps/business-services/index.mdx). Select one or more database instances on the on-prem databases list page and choose **Add to Business Service** from the **Actions** dropdown menu.

<ThemedImage
  alt="Add to business application"
  sources={{
    light: useBaseUrl('/assets/images/on-premise-databases/add-to-business-app-19.06-light.png'),
    dark: useBaseUrl('/assets/images/on-premise-databases/add-to-business-app-19.06-dark.png'),
  }}
/>

Select a **Business Service name** from the dropdown list or click the **plus icon** to create a new Business Service. Click **Add selected On-Prem Database to Business Service** to confirm.

<ThemedImage
  alt="Add to business application dropdown"
  sources={{
