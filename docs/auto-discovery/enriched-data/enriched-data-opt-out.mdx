You can disable the data Normalization and Enrichment Service if you no longer want Device42 to enrich data from device discoveries. This page walks through the steps to opt out and opt back in.

:::note
If you rediscover previously enriched data, you will lose that data until you opt back in to the Enriched Data cloud service.
:::

## Disable the Enriched Data Service

To opt out of the Enriched Data service:

1. Select **Tools > Cloud Services** to display the Cloud Services page. This page shows the status of your cloud services.

   <ThemedImage
     alt="Cloud Services page"
     sources={{
       light: useBaseUrl('/assets/images/enrichai-opt-out/cloud-services-button-light.png'),
       dark: useBaseUrl('/assets/images/enrichai-opt-out/cloud-services-button-dark.png'),
     }}
     style={{ width: '90%' }}
   />

2. Find the **EnrichAI** entry showing **Enabled** and click **Edit**.

   <ThemedImage
     alt="View integrations page"
     sources={{
       light: useBaseUrl('/assets/images/enrichai-opt-out/view-integration-light.png'),
       dark: useBaseUrl('/assets/images/enrichai-opt-out/view-integration-dark.png'),
     }}
   />

3. In the Edit view, click **Disable**.

   <ThemedImage
     alt="Disable button"
     sources={{
       light: useBaseUrl('/assets/images/enrichai-opt-out/disable-button-light.png'),
       dark: useBaseUrl('/assets/images/enrichai-opt-out/disable-button-dark.png'),
     }}
   />

4. In the **Disable EnrichAI** dialog, select **Disable** to confirm. Select **Cancel** to return without changes.

   <ThemedImage
     alt="Disable prompt"
     sources={{
       light: useBaseUrl('/assets/images/enrichai-opt-out/prompt-light.png'),
       dark: useBaseUrl('/assets/images/enrichai-opt-out/prompt-dark.png'),
     }}
     style={{ width: '65%' }}
   />

5. The **EnrichAI** status is now **Disabled**. Click **Save** in the bottom-right corner of the page.

   <ThemedImage
     alt="Save button"
     sources={{
       light: useBaseUrl('/assets/images/enrichai-opt-out/disabled-save-light.png'),
       dark: useBaseUrl('/assets/images/enrichai-opt-out/disabled-save-dark.png'),
     }}
   />

## Re-Enable the Enriched Data Service

Follow the same steps to opt back in to the Enriched Data service. After re-enabling, rerun your discovery jobs to ensure your data is enriched.
