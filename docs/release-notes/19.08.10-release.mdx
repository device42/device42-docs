*Release Date: February 25, 2026.*

## Updating Device42 to Version 19.08.10

Upgrading to 19.08.10 is possible from 19.05.10 or above. If you need to first upgrade to Version 19.05.10, please see those release notes for details. A link will also be provided when requesting an upgrade. We recommend making a backup of your system before any upgrades.

For up-to-date information, notifications of outages or maintenance, visit the status page at: [https://device42.freshstatus.io/](https://device42.freshstatus.io/).

The 19.08.10 MA release contains the following features:

## Business Service Enhancements

It is now possible to link business services to other related business services within the business service editor.

![Business service linking example](/assets/images/release-notes/business-service-example.png)

Application component templates can now be associated with software components.

![Application component and software component association](/assets/images/release-notes/app-comp-software-comp-example.png)

- Improved IIS application modeling and connection creation on servers with multiple w3wp processes.
- Improved agent-based application component updates from a template.

## Enrichment Service Enhancements

Enrich will now upload more records for enrichment, as the accuracy threshold for which software is enriched was too high. This will lead to significantly more software records being enriched over time.

## Security Updates

Across many aspects of Device42, security has been enhanced, including updating core OS dependencies, used libraries, and appliance configuration.
