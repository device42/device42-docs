This page is for administrators who want to connect Device42 to the other tools in their IT ecosystem, such as ITSM platforms, secret managers, alerting tools, and BI applications. Use it as your starting point to find the right integration and follow its dedicated setup guide.

Device42 integrates with many popular third-party ITSM, configuration management, SIEM, and IPAM tools. Each integration keeps your infrastructure data consistent across systems, so the CMDB you build through discovery stays useful everywhere your teams work.

## How to Use This Section

Each integration has its own page with setup steps, prerequisites, and configuration options. Find the integration you need in the catalog below, or browse the full list in the sidebar on the left.

Before you start, most integrations require:
- Administrator access to both Device42 and the third-party system
- Credentials or API keys for the third-party system
- Network connectivity between Device42 and the third-party system

Check the individual integration page for its exact prerequisites.
